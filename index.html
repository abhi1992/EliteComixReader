Elite Comix Reader

A simple java based comic book reader.

Download this project as a .zip file Download this project as a tar.gz file

A simple java based comic book reader. Capable of reading .cbr, .cbz, .zip, .rar files containing image files. Image formats supported include .jpg, .png, .gif, .bmp, .wbmp.
